# GhidraWTF? (What The Function?)
GhidraWTF? is a Ghidra plugin that offers an LLM-assisted understanding of decompiled code

## Usage (Ghidra Script)

This repo currently provides a **PyGhidra script** in `ghidrawtf.py`.

- **Install**: copy `ghidrawtf.py` into your Ghidra scripts directory (or add this repo folder to the Script Manager script paths).
- **Run**: in Ghidra, open the Script Manager and run the script, or use the menu entry:
  - `Tools -> Expl(AI)n Current Function`

## Configuration

Edit the script and add your API key
